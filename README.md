# 🏠 OshEstate - Платформа недвижимости

## ⚡ Быстрый старт

### Шаг 1: Запустить веб-сервер

**Вариант 1: Через батник (Рекомендуется для Windows)**
- Двойной клик по файлу `run_server.bat`
- Откроется окно командной строки
- Сервер запустится на `http://localhost:8000`

**Вариант 2: Через PowerShell**
```powershell
cd "c:\Users\User\Desktop\Новая папка\Квант"
python -m http.server 8000
```

**Вариант 3: Через Git Bash или другой терминал**
```bash
cd /c/Users/User/Desktop/Новая\ папка/Квант
python -m http.server 8000
```

### Шаг 2: Открыть в браузере

После запуска сервера, откройте браузер и перейдите по адресу:
```
http://localhost:8000
```

## 🔐 Регистрация и авторизация

### Регистрация

1. **Перейти**: http://localhost:8000/register.html
2. **Заполнить форму:**
 - Имя
- Номер телефона (WhatsApp)
   - Email
   - Пароль (минимум 6 символов)
3. **Нажать**: "Получить код"
4. **Ввести код подтверждения:** `1234` (для тестирования)
5. **Нажать**: "Подтвердить и войти"
6. **Готово!** Вы перенаправлены на главную страницу с авторизацией

### Вход

1. **Перейти**: http://localhost:8000/login.html
2. **Ввести:**
   - Email от зарегистрированного аккаунта
   - Пароль
3. **Нажать**: "Войти"

## 📝 Создание объявления

Только для авторизованных пользователей!

1. **Быть авторизованным** (см. регистрация выше)
2. **Перейти**: http://localhost:8000/add.html
   - ИЛИ нажать кнопку "+ Подать" в навбаре
3. **Заполнить форму:**
   - **Тип сделки**: Продажа или Аренда
   - **Категория**: Квартира, Дом, Участок, Коммерция
   - **Комнат**: Количество комнат (или 0 для земли)
   - **Цена**: В сомах
   - **Площадь**: В м²
   - **Адрес**: Можно вводить текст (автодополнение) или кликнуть на карте
   - **Фотографии**: Загрузить макс 5 фото
   - **Описание**: Описание объекта
4. **Нажать**: "Опубликовать"
5. **Объявление отправлено на проверку!** Администратор должен одобрить

## 👤 Профиль пользователя

### Управление профилем
- **Перейти**: http://localhost:8000/profile.html
  - ИЛИ нажать на имя в навбаре "Профиль"
- **Функции:**
  - Просмотр активных объявлений
  - Просмотр объявлений на проверке
  - Редактирование объявления (цена, адрес, описание)
  - Удаление объявления
  - Загрузка аватара
  - Изменение имени
  - Выход из аккаунта

### Избранные объявления
- **Перейти**: http://localhost:8000/favorites.html
  - ИЛИ нажать "❤️ Избранное" на странице профиля
- **Сохранить объявление**: Кликнуть на сердечко на карточке
- **Просмотр**: Все избранные объявления в одном месте

## 🐛 Отладка

Если что-то не работает:

1. **Откройте консоль браузера:**
   - Windows/Linux: F12 → Console
   - Mac: Cmd + Option + J

2. **Проверьте:**
   - Нет ли красных ошибок
   - Сервер работает (http://localhost:8000)
   - Firebase подключен (смотрите console.log)

3. **Проверьте сетевые запросы:**
   - F12 → Network
   - Все запросы должны вернуть статус 200 или 304

4. **Очистите кэш:**
   - Ctrl + Shift + Delete (Windows)
   - Cmd + Shift + Delete (Mac)
